Campaign Share

# 소개 
자신만의 아이디어로 캠페인을 만들어 
사회에 선한 영향력을 줄 수 있는 서비스입니다.

# 기능
* 사용자는 진행중인 캠페인을 한눈에 알아볼 수 있습니다.  
* 사용자는 자신만의 새로운 캠페인을 만들거나 남이 만든 캠페인에 참여할 수 있습니다.
